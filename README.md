## Setup & Installation

1. Install **Arduino IDE** or **PlatformIO**  
2. Install ESP32 board support  
3. Install required libraries:  
   - Adafruit SSD1306  
   - Adafruit GFX  
   - TinyGPS++  
   - WiFi / WebSocket / MQTT libraries  
4. Clone this repository  
5. Edit config values (WiFi, server URL, etc.)  
6. Upload to ESP32
